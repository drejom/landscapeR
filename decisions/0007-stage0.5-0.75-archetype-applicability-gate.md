# 0007 — Stage 0.5/0.75 as the real-data applicability gate

**Stage:** 0.5 / 0.75 / 2
**Status:** accepted
**Date:** 2026-07-12

## Context

The initial Stage 2 estimator fits a one-dimensional quasi-potential on one
selected target biological axis. This is appropriate for ordered one-dimensional
contrasts, but biological data can instead be continuous, single-well,
branching, or genuinely multi-axis. *Pogona* sex-determination trajectories are
a motivating bifurcating case.

Without an explicit applicability gate, a real-data analysis can select a
component and produce a visually persuasive one-dimensional landscape even when
that representation is unsupported. This would make disease-specific or
ordered-transition assumptions appear to be general package philosophy.

Stage 0.5 is planned to compare an observed signature with synthetic trajectory
archetypes. Stage 0.75 is planned to assess whether a real dataset lies within
the validated synthetic regime. Their contracts are currently deferred and
unexported (Issue #5); this ADR defines their scientific role, not their
implementation algorithm.

## Options considered

| Option | Key property | Disqualifier or concern |
|---|---|---|
| Run Stage 2 first; classify the result afterwards | Smallest immediate workflow | Lets an unsupported estimator create the evidence used to justify itself. |
| Require analyst judgement alone | Flexible | Not reproducible, scalable, or auditable. |
| Use Stage 0.5/0.75 as an applicability gate for real-data claims | Separates geometry classification, fit assessment, and estimation | Requires synthetic reference archetypes and deferred implementation work. |
| Build a universal 2D/branching estimator now | Could model bifurcation directly | No chosen algorithm or Stage 0 evidence; would violate evidence-first selection. |

## Criteria

- Never force a one-dimensional state-transition conclusion on branching,
  multi-axis, or unsupported data.
- Keep the package biologically general: disease, sex determination,
  development, treatment, and other contrasts are instances, not assumptions.
- Make the eligibility of a Stage 2 claim reproducible and provenance-recorded.
- Do not select a future 2D/bifurcation algorithm before its synthetic-control
  criteria and benchmark ladder exist.

## Evidence

No Stage 0 archetype-control results yet exist. This is an architectural and
claim-scope decision made before implementation. The existing Stage 2 estimator
uses one component at a time, so it cannot represent a bifurcation without
flattening it.

## Decision

**Chosen:** Stage 0.5 and Stage 0.75 are required applicability gates for
confirmatory real-data Stage 2 claims.

Stage 0.5 classifies the observed geometry against a synthetic archetype
library. Stage 0.75 assesses whether the observation lies within a validated
synthetic regime. The one-dimensional Stage 2 estimator may support a
confirmatory claim only when the selected target biological axis is classified
as an eligible ordered one-dimensional archetype and passes fit assessment.

A single-well/continuous result is a valid no-transition conclusion. A
branching or multi-axis result is explicitly outside the current estimator's
scope, not evidence to flatten into a one-dimensional barrier. Exploratory
analyses must carry that status in their output and provenance.

## Consequences

- Implement the core Stage 0 end-to-end controls first; do not implement the
  deferred 0.5/0.75 contracts merely to satisfy the gate.
- When 0.5/0.75 implementation starts, their synthetic library must include
  one-dimensional ordered, single-well/continuous, and branching/multi-axis
  archetypes.
- A future 2D or bifurcation Stage 2 strategy requires a separate ADR and
  Stage 0 bifurcation-control ladder before use on *Pogona* or other real data.
- Real-data one-dimensional results produced before the gate exists are
  exploratory rather than confirmatory.

## Review trigger

Revisit when Stage 0 archetype controls show that the proposed signatures or
fit assessment cannot reliably distinguish eligible one-dimensional trajectories
from branching/multi-axis cases, or when a candidate 2D estimator has
predeclared recovery criteria.
